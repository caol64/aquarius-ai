# AquariusAI

`AquariusAI`的设计理念旨在让您更方便地运用各种开源模型，以辅助增强日常工作的效率及提高生活质量，同时也解决了使用商用模型可能带来的隐私泄露和费用成本的担忧。`AquariusAI`本身不提供任何`AI`模型，所以您需要自行准备所需的模型。

## 特性

- 文生文
  - 适用于所有`Ollama`模型（Llama 3、Phi 3, Mistral, Gemma 2等）
  - 知识库和`RAG`
- 文生图
  - 适用于`CoreML`格式的`Stable Diffussion`模型（包括`XL`模型）
  - AI Upscaling

不断增加中...

## 最低系统要求

- macOS 14.0 Sonoma
